# To-Do List Web App

A simple To-Do List app created with **HTML**, **CSS**, and **JavaScript**.  
This project allows users to add and remove tasks easily.

## Features
- Add tasks
- Remove tasks
- Clean, responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone this repository:
  
3. Open `index.html` in your browser to view the app.
